Securonis Linux

Securonis Linux is a privacy- and security-focused Linux distribution based on Debian. It includes a preconfigured tool that forces all incoming and outgoing connections through the Tor network. The system comes preloaded with various privacy and security tools, along with some custom-developed utilities. Securonis Linux uses the MATE and GNOME desktop environment. It can be run in live mode or installed permanently using the Calamares installer. The goal of Securonis is to provide a secure system suitable for everyday use.

About Securonis

Securonis GNU/Linux - Privacy-Focused Distro Based on Debian Trixie

Securonis GNU/Linux is a lightweight and high-performance operating system based on Debian, prioritizing privacy and security.

At its core lies a custom tool named Seconionis, which forces all incoming and outgoing traffic through the Tor network. This tool also modifies kernel and network configurations to minimize the risk of data leakage.

Securonis comes pre-installed with the I2P Router, a graphical user interface, and a preconfigured browser—making access to the I2P network seamless and user-friendly.

The system includes a wide range of encryption, security, and data wiping tools. DNS queries are encrypted via DNSCrypt-Proxy, ensuring secure DNS resolution.

With the System Hardening Script, kernel and network settings are fortified to protect against unauthorized access.

A powerful emergency feature called Panic Mode is also included:

  • Physical Security Mode - defends against physical threats.
  • Paranoia Mode - disconnects all communications and maximizes kernel security.
  • Nuke the System - securely wipes the disk and irreversibly destroys all data.

Securonis uses the MATE and GNOME desktop environment and the Calamares installer for easy setup.

Our mission is to deliver a secure, minimalistic, and privacy-respecting computing experience for everyday use.

System-Wide Anonymity and Privacy Features

Seconionis

  • Routes all system traffic through the Tor network using a transparent proxy.
  • Written in Shell with a Python-based GUI.
  • Works as a systemd service for background operation.
  • Includes MAC address changer, Tor identity renewal, and trace cleaner.

I2P Router Menu

  • GUI tool for easy I2P routing on Linux.
  • Pre-configured Firefox for I2P browsing.

DNSCrypt-Proxy

  • Encrypts DNS requests to prevent tracking.
  • Pre-configured and ready to use.

Digital fingerprint Protection

  • System and browser tweaks to reduce digital fingerprint and tracking.

Security Mechanisms

OpenMammoth

  • Customized network defense and simulation toolkit for Securonis.

System Hardening Script

  • Uses custom kernel tweaks to improve system security and resilience.

Panic Mode

  • System Status Checks, Leak Tests, Trace Cleaner
  • Physical Security Mode – disables ports, blocks USB/Bluetooth, restricts permissions.
  • Paranoia Mode – disconnects from networks, enables maximum kernel-level protection, isolates system.
  • Nuke2System – securely overwrites disk, wipes RAM and metadata, irreversibly destroys data.

Warning: The Securonis System Wipe Mode has been developed to prevent data theft in the event of virus infections, cyberattacks, or system compromise by malicious entities. This feature is intended solely as a security measure to protect your personal data in emergency situations. It is not designed for misuse.

Why Securonis?

My goal was simple: to create a minimal, fast, secure, and privacy-respecting Linux distribution.

In today's world, it's hard to find a Linux distro that can be installed to a hard drive and is truly designed for daily use while also prioritizing privacy and security.

Securonis does exactly that. You won't need to rely on virtual machines or flash drives anymore. No more messing around with complex Tor routing setups — everything is handled for you through tools developed specifically for Securonis.

It offers a safe and secure computing environment for everyday use. Whether it's a hardened browser experience, system protection, or trace removal — Securonis has you covered.

You can also run it in Live Mode, meaning you don't need to install it right away. You can try it out or use it portably — the choice is yours.

Designing Securonis was a fun journey, though not always easy. I truly hope you enjoy using it as much as I enjoyed building it!
